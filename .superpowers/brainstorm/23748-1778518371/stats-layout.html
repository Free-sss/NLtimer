统计页面布局方案

选择你偏好的整体页面结构

A

分段滚动页

单页垂直滚动：今日摘要色块 → 近7天柱状图 → 近30天柱状图 → 活动排行 → 标签分布。类似 Tomato 参考。简洁直接。

B

Tab 切换页

顶部 Tab 栏（今日 / 本周 / 本月 / 全部），每个 Tab 展示对应时间维度的完整统计面板。信息密度更高，切换有动画。

C

仪表盘卡片网格

2列网格排列多个圆角卡片（总时长、完成次数、番茄钟数、最常活动等），每个卡片带数字动画。底部放图表区。信息密度最高。
