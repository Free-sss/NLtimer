修正：移除 Google Sans

字体选项精简为 Figtree + System Default 两种

🔤 Fonts 枚举

// core:designsystem/Fonts.kt

enum class Fonts {
    FIGTREE,
    SYSTEM_DEFAULT,
}

// core:designsystem/enumExt.kt — 字体映射

fun Fonts.toFontRes(): Int? = when (this) {
    Fonts.FIGTREE        -> R.font.figtree
    Fonts.SYSTEM_DEFAULT -> null       // null → FontFamily.Default
}

fun Fonts.toDisplayString(): String = when (this) {
    Fonts.FIGTREE        -> "Figtree"
    Fonts.SYSTEM_DEFAULT -> "System Default"
}

📁 影响范围

文件 变更
Fonts.kt 仅 FIGTREE + SYSTEM_DEFAULT
enumExt.kt 移除 GOOGLE_SANS 分支
Theme.kt 默认值保持不变 (FIGTREE)
资源文件 仅复制 figtree.ttf，不复制 google_sans_flex.ttf
⚠️ 检查 Figtree 许可：Figtree 也需确认 Momentum 中的源文件是否附带 OFL 声明。如有疑问可改用系统自带字体或从 Google Fonts 官方渠道下载。
