📱 界面组成说明

1️⃣ 顶部导航栏 (TopAppBar)

  • 返回按钮 ← ：返回上一页
  • 标题："标签管理"
  • 操作按钮 🗑️：批量删除模式切换（可选）

2️⃣ 分类卡片 (CategoryCard)

  • 分类标题：显示分类名称（"默认"、"优先级"、"状态"等）
  • 操作菜单 ⋮：重命名、删除、排序等操作
  • 标签列表：FlowRow 横向排列的 TagChip
  • 添加按钮 +：在该分类下新建标签

3️⃣ 标签芯片 (TagChip)

  • 样式：圆角胶囊形状，带背景色
  • 颜色：根据 tag.color 字段动态设置（或使用分类主题色）
  • 交互：
    • 单击 → 打开编辑对话框
    • 长按 → 显示操作菜单（移动、删除）

4️⃣ 底部入口

  • + 增加标签分类：虚线边框卡片，点击弹出新建分类对话框

5️⃣ 浮动按钮 (FAB)

  • 位置：右下角
  • 功能：展开更多操作菜单（排序方式、导入/导出等）
