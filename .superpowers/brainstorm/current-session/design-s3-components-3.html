⑨ SettingsCard 自动圆角

现有：3个 shape 函数
leadingItemShape()
middleItemShape()
endItemShape()
detachedItemShape()
需要3+1个函数 + 手动判断位置

融入后：GroupCard 组件
GroupCard
items: List<Composable>
cornerStrategy: auto(基于index/size)
// 内部自动算：
size==1 → 全圆角
index==0 → 上圆下直
index==last → 上直下圆
else → 直角(6dp)
一个组件替代3个函数 + 手动判断
圆角值跟随 styledCorner() 三档联动
替代 leadingItemShape/middleItemShape/endItemShape，保留 detachedItemShape 用于单独项
