三个核心设计要点

1. 空白容器，零耦合

DebugRoute 是一个完全干净的 Compose 容器，不依赖任何 ViewModel、Repository 或业务数据。纯 UI 渲染。

2. 快速切换组件

通过顶部 Dropdown / Tab 选择器，即时切换不同的 UI 组件进行预览，支持实时热重载。

3. 可控制入口

通过 BuildConfig 或 DataStore 开关控制导航入口的显隐，发布版本完全隐藏。
