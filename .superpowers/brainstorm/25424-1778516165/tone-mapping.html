色调族映射偏好

在含蓄分层（container 级）的前提下，primary / secondary / tertiary 如何分配给不同状态

方案 1 — 状态驱动
活跃行为 ← primary container（当前种子色）
已完成 ← secondary container（互补色调）
空白待填 ← tertiary container（第三色调）
锁定/未来 ← surface variant（中性）
侧边栏/当前行指示 ← primary（非 container，稍重）

三种色调全部用于行为状态，侧边栏用 primary 标识导航焦点

方案 2 — 区域驱动
活跃行为 + 侧边栏当前 ← primary container
已完成 + 侧边栏活跃 ← secondary container
空白 / 锁定 ← surface variant（保持中性）
白金成就 / 特殊标记 ← tertiary container（保留给亮点）

primary/secondary 同时驱动行为状态和对应侧边栏区域，tertiary 保留给高光时刻，空白保持中性
